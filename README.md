### Colors
Update colors in `tailwind.config.js`:
```javascript
colors: {
  'dark-slate': '#1E4D4F',
  'linen': '#F4EDE4',
```

- Modify product data in `src/data/products.js`

- Adjust animation timing in Framer Motion components
- Modify AOS settings in `App.js`
